# 🛒 FarmEasy - E-Commerce Website

FarmEasy is a responsive front-end e-commerce web application developed using HTML, CSS, JavaScript, Bootstrap, and jQuery. The project simulates an online grocery shopping platform where users can browse products, add items to the cart, proceed to checkout, and place orders.

## 🚀 Features

- Product listing with pricing details
- Add to Cart functionality
- Shopping Cart management
- Checkout and Order Summary pages
- Customer Billing and Shipping Information
- Responsive user interface
- Interactive navigation and product sections
- Simulated online shopping experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 4
- Font Awesome

## 🎯 Key Functionalities

- Browse fresh fruits and vegetables
- Add products to shopping cart
- Update or remove cart items
- Enter billing and shipping details
- Review order before payment
- Simulated checkout workflow

## 💡 Learning Outcomes

- Front-End Web Development
- Responsive Web Design
- DOM Manipulation using JavaScript
- Shopping Cart Implementation
- Form Validation
- User Interface Design
